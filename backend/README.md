# AI Learning Assistant Backend

## 技术栈

- Spring Boot 3
- MyBatis Plus
- MySQL
- Redis
- Lombok

## 当前已完成

- Maven 项目基础结构
- MyBatis Plus 基础配置
- 自动填充时间字段
- 逻辑删除基类
- 与数据库表对应的第一批实体类
- 基础 Mapper 接口
- 统一返回体与全局异常处理
- 注册、登录、获取当前用户信息接口
- JWT 鉴权拦截基础能力
- 文本资料新增、分页、详情、删除接口
- 文件上传与 `txt/pdf/docx` 解析入库能力
- AI 总结生成、生成记录落库、AI 笔记保存
- AI 出题、题集查询、练习开始、练习提交与结果查询

## 建议下一步

1. 为前端补题集列表、答题页和练习结果页。
2. 增加错题本与复习记录联动。
3. 把状态字段提取成统一枚举，减少硬编码。
4. 最后增加接口文档和部署配置。
